# dotfiles

## Description

This is the git repository for my personal dotfiles for bash, vim, git and tmux. At the moment I'm using the [Solarized](https://ethanschoonover.com/solarized/) colors for pretty much everything.

## Windows Subsystem for Linux

- Windows 11 + Windows Subsystem for Linux 2 + Ubuntu
- Windows Terminal App
- Font [JetBrainsMono Nerd Font](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.3.0/JetBrainsMono.zip)
- Colorscheme [Solarized Dark Patched for Windows Terminal](https://github.com/Lucky-Loek/patched-solarized-dark-windows-terminal)

## Installation

I'm working on some scripts to automate backing/cleaning up the existing configuration files, download and/or clone necessary plugins/files and linking them accordingly.
